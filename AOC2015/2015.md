# Advent of Code 2015 - Algorithm Summary

## Day 1: Character Counting and Linear Tracking

Simple character counting for floor calculation with position tracking for basement detection.

## Day 2: Mathematical Calculation and Optimization

Geometric calculations for surface area and ribbon requirements with minimum value selection.

## Day 3: 2D Coordinate Tracking with Set Operations

Movement simulation on 2D grid using coordinate tracking and visited house counting.

## Day 4: Cryptographic Hash Generation with Brute Force

MD5 hash generation with iterative searching for specific leading zero patterns.

## Day 5: String Validation with Pattern Matching

String analysis using character counting, pattern detection, and constraint validation.

## Day 6: 2D Grid State Management

Grid-based light control simulation with toggle/brightness operations and state tracking.

## Day 7: Circuit Logic Simulation with Memoization

Boolean logic circuit evaluation using recursive parsing with caching optimization.

## Day 8: String Processing with Escape Sequence Handling

Character encoding/decoding analysis using regex and escape sequence processing.

## Day 9: Traveling Salesman Problem with Permutations

Graph traversal optimization using permutation generation for shortest/longest path finding.

## Day 10: Iterative String Transformation

Look-and-say sequence generation using iterative string building and pattern recognition.

## Day 11: String Increment with Constraint Validation

Password generation using string increment operations with multiple validation rules.

## Day 12: JSON Parsing and Tree Traversal

JSON document analysis using recursive traversal with conditional filtering and number extraction.

## Day 13: Seating Optimization with Permutations

Circular arrangement optimization using permutation generation and happiness calculation.
