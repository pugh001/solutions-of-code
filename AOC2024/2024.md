# Advent of Code 2024 - Algorithm Summary

## Day 1: Sorting and Frequency Counting
Two lists comparison using sorting and frequency mapping for distance calculation and similarity scoring.

## Day 2: Array Validation with Dampener
Linear scan validation for increasing/decreasing sequences with single element removal tolerance.

## Day 3: Regular Expressions
Pattern matching with regex to extract and process mul() instructions with conditional filtering.

## Day 4: 2D Grid Word Search
Multi-directional word search in 2D character grid using direction vectors and pattern matching.

## Day 5: Topological Sorting
Dependency graph construction and topological sorting for ordering validation and correction.

## Day 6: 2D Grid Simulation with Cycle Detection
Guard movement simulation on 2D grid with obstacle detection and loop/cycle identification.

## Day 7: Recursive Backtracking
Expression evaluation using recursive operator combination generation and left-to-right evaluation.

## Day 8: 2D Grid Coordinate Geometry
Antenna positioning with coordinate calculations for antinode placement using distance vectors.

## Day 9: Disk Defragmentation Simulation
Block-based data structure manipulation for disk compaction with gap handling algorithms.

## Day 10: 2D Grid BFS/DFS Pathfinding
Breadth-first and depth-first search on height-constrained 2D grid for trail counting.

## Day 11: Dynamic Programming with Memoization
Stone transformation using recursive approach with memoization for exponential growth simulation.

## Day 12: 2D Grid Flood Fill with Perimeter Calculation
Region detection using flood fill algorithm with area and perimeter/sides calculation.

## Day 13: Linear Algebra System Solving
System of linear equations solving using mathematical formulas for prize optimization.

## Day 15: 2D Grid Movement Simulation

## Day 16: 2D Grid Dijkstra's Algorithm

## Day 17: Virtual Machine Simulation

## Day 18: 2D Grid BFS with Dynamic Obstacles

## Day 19: String Pattern Matching with Memoization

## Day 20: 2D Grid BFS with Shortcut Detection
